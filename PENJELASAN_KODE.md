# Penjelasan Kode Borneo Scholar

ZIP ini berisi project yang sama, tetapi file Dart di folder `lib/` sudah diberi komentar singkat dan jelas.

## Cara pakai
1. Extract ZIP ini.
2. Buka folder project di VS Code.
3. Baca file Dart di folder `lib/`.
4. Komentar yang diawali `//` adalah penjelasan fungsi kode.
5. Jalankan seperti biasa dengan:
   ```bash
   flutter pub get
   flutter run
   ```

## Catatan
Komentar tidak mengubah fungsi aplikasi. Fungsinya hanya membantu memahami kode dan memudahkan saat presentasi.
